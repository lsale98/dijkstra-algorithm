Pathfinder

Dijkstra's Algorithm

The goal of the program is to find the shortest path from start to finish using the Dijkstra algorithm.

How to use

  1. Program initialize grid where the start and finish are set
  2. (Optional) Left-clicking on a node sets the wall if it was not set in that place, otherwise the wall is removed from the node
  3. Clicking on the button Visualize Dijkstra's Algorithm starts the algorithm where you can see the process of searching for the shortest path
